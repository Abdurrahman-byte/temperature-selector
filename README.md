 🌡️ Temperature Selector (Tkinter GUI)

A simple Python GUI application built with Tkinter that allows users to select a temperature value using a slider and display the selected value.



📌 Overview

The Temperature Selector is a beginner-friendly Tkinter project that demonstrates how to use the `Scale` widget to collect numeric input from users.

Users can move a slider to choose a temperature between 0°C and 100°C and then submit their selection using a button.

This project introduces graphical controls, event handling, and user interaction in desktop applications.



 Features

* 🌡️ Temperature slider (0°C - 100°C)
* 🎚️ Interactive Scale widget
* 🖱️ Button-triggered action
* 📺 Displays selected temperature
* 🎨 Customized slider appearance
* 🖼️ Simple graphical interface



  Technologies Used

* Python 3
* Tkinter (GUI Framework)



   How to Run

1. Clone the repository

```bash
git clone https://github.com/Abdurrahman-byte/temperature-selector.git
```

2. Navigate to the project folder

```bash
cd temperature-selector
```

3. Run the application

```bash
python main.py
```



  How to Use

1. Launch the application.
2. Move the slider to select a temperature.
3. Click the **Submit** button.
4. The selected temperature will be displayed in the console.

Example Output

```text
The temperature is: 35 degree C
```
